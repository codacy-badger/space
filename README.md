# Galactium Space

Galactium Space is light library to provide extend functionality for Galactium projects.

Please note: the library is under development.

### Requirements

 * PHP 7.1 or above
 * Phalcon 3.3

## Built With

* [Phalcon](https://phalconphp.com) - The web framework used
